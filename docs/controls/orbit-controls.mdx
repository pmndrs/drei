Controls that orbit around a target.
Wraps THREE's [OrbitControls](https://threejs.org/docs/#examples/en/controls/OrbitControls).

### Basic usage

```jsx
<OrbitControls />
```

### With options

```jsx
<OrbitControls makeDefault enableDamping dampingFactor={0.05} />
```
