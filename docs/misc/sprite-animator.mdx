Animates a sprite sheet texture across a plane.
Supports auto-play, pause, loop, and frame control via ref.

### Basic sprite animation

```jsx
<SpriteAnimator
  spriteDataset={spriteData}
  textureImageURL="/spritesheet.png"
  fps={24}
  autoPlay
  loop
/>
```
