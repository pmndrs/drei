Allows selecting/unselecting objects by clicking on them.
Supports multi-select (shift key) and box selection.

### Basic usage

```jsx
<Select box multiple onChange={console.log}>
  <Foo />
  <Bar />
</Select>

function Foo() {
  const selected = useSelect()
}
```
