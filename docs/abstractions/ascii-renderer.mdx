Abstraction of three's [AsciiEffect](https://threejs.org/examples/?q=as#webgl_effects_ascii).
Creates a DOM layer on top of the canvas and renders the scene as ASCII characters.

### Basic usage

```jsx
<Canvas>
  <AsciiRenderer />
</Canvas>
```
