Builds a Bounding Volume Hierarchy for faster raycasting.
Useful for scenes with complex geometry.

### Basic usage

```jsx
<Bvh>
  <mesh>
    <torusKnotGeometry args={[1, 0.3, 100, 100]} />
  </mesh>
</Bvh>
```
