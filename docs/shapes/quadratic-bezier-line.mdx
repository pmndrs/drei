Renders a THREE.Line2 using THREE.QuadraticBezierCurve3 for interpolation.

### Basic usage

```jsx
<QuadraticBezierLine
  start={[0, 0, 0]}
  mid={[1, 1, 0]}
  end={[2, 0, 0]}
  segments={20}
/>
```
