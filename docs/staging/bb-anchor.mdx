Anchors children to the bounding box of the parent mesh.

### Basic usage

```jsx
<mesh>
  <boxGeometry />
  <BBAnchor anchor={[1, 1, 0]}>
    <Html>Top Right</Html>
  </BBAnchor>
</mesh>
```
