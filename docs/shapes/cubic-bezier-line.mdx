Renders a THREE.Line2 using THREE.CubicBezierCurve3 for interpolation.

### Basic usage

```jsx
<CubicBezierLine
  start={[0, 0, 0]}
  midA={[1, 1, 0]}
  midB={[2, 1, 0]}
  end={[3, 0, 0]}
  segments={20}
/>
```
