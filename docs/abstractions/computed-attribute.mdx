Create and attach a buffer attribute declaratively.
Used exclusively as a child of a BufferGeometry.

### Basic usage

```jsx
<sphereGeometry>
  <ComputedAttribute
    name="my-attribute-name"
    compute={(geometry) => {
      // ...someLogic
      return new THREE.BufferAttribute([1, 2, 3], 1)
    }}
    usage={THREE.StaticReadUsage}
  />
</sphereGeometry>
```
