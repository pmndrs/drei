Loads a cube texture (6 images) for use as environment maps.

### Basic usage

```jsx
<CubeTexture files={['px.png', 'nx.png', 'py.png', 'ny.png', 'pz.png', 'nz.png']} path="cube/">
  {(texture) => <primitive object={texture} attach="background" />}
</CubeTexture>
```
