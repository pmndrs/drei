Renders a THREE.Line2 or THREE.LineSegments2 (depending on the value of `segments`).

### Basic usage

```jsx
<Line
  points={[[0, 0, 0], [1, 0, 0], [1, 1, 0]]}
  color="black"
  lineWidth={1}
  segments={false}
  dashed={false}
  vertexColors={[[0, 0, 0], [1, 0, 0]]} // Optional RGB for each point
/>
```
