Renders a THREE.Line2 using THREE.CatmullRomCurve3 for interpolation.

### Basic usage

```jsx
<CatmullRomLine
  points={[[0, 0, 0], [1, 1, 0], [2, 0, 0]]}
  closed={false}
  curveType="centripetal"
  tension={0.5}
  color="black"
  lineWidth={1}
/>
```
