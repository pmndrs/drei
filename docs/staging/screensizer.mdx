Wraps children in an `Object3D` and attempts to scale from
world units to screen units * scale factor.

For example, this will render a box of roughly 1x1 pixel size,
independently of how far the camera is.

```jsx
<ScreenSizer>
  <Box />
</ScreenSizer>
```
